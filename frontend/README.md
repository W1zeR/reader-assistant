# reader-assistant-frontend

Allows users to save and share quotes from literary works

## Running

1. Run `npm run dev`
2. Visit `http://localhost:3000`

## Architecture

- Next.js
- NextAuth.js
- Axios
- Zod
- Tailwind CSS
- NextUI

## Features

- Dark theme
- JWT refresh
- Search with debounce
